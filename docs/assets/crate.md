# crate

Wooden slat crate heaped with produce.

## Look

Plank slats over a dark inner box; heaped with faceted produce balls
(`fruit` straw-gold/tan/plum, `plums` mauve) or leafy cone bunches
(`greens`).

## Dimensions & origin

≈ 1.1 wide × 0.9 tall × 0.8 deep. Origin at ground-center.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | heap arrangement |
| `produce` | string | `fruit` | `fruit` \| `greens` \| `plums` \| `empty` |

## Animations

`animations: none` — static asset.
