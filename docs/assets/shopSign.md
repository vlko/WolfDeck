# shopSign

Wooden post with a crossbar and a blank framed sign hanging on two straps.

## Dimensions & origin

≈ 1.1 wide × 2.7 tall. Origin at ground-center of the post.

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |
| `color` | color | `barkBrown` |

## Animations

| name | description |
| ---- | ----------- |
| `swing` | the hanging board sways gently in the breeze, with a faint high-frequency tremble (default) |
| `none` | still |
