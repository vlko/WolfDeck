# streetlamp

Charcoal lamppost with a warm glowing lantern ball.

## Look

Slim charcoal pole and cap, faceted `sunGold` lantern ball with a soft
transparent halo disc. Blob shadow.

## Dimensions & origin

≈ 3.2 tall. Origin at ground-center of the pole.

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |

## Animations

| name | description |
| ---- | ----------- |
| `glow` | the halo breathes and flickers faintly, like warm gaslight (default) |
| `none` | static |
