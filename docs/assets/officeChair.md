# officeChair

Round swivel chair — disc base, stem, round seat, circular backrest, per
the reference sheet.

## Dimensions & origin

≈ 0.8 wide × 1.25 tall. Origin at ground-center.

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |
| `color` | color | `tanBrown` |

## Animations

| name | description |
| ---- | ----------- |
| `swivel` | rocks gently left and right around its stem |
| `none` | static (default) |
