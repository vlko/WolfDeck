# churchTower

White village church with a slender clock tower and teal spire — the
Námestovo skyline landmark for back rows.

## Look

Long `paperWhite` nave with a `waterTeal` gabled roof and three dark arched
window slits; slender white tower on the left with a `cream` clock face in a
teal ring (seeded hand positions), a dark belfry slit, teal eaves collar and
a tall 8-sided faceted teal spire topped with a `strawGold` orb and cross.
Plank door at the tower base. Blob shadow.

## Dimensions & origin

≈ 4.6 wide × 6.6 tall (spire tip) × 1.7 deep at defaults. Origin at
ground-center; the tower stands left of origin, the nave right.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | clock hands, facet jitter |
| `wallColor` | color | `paperWhite` | nave + tower walls |
| `spireColor` | color | `waterTeal` | spire, roofs, clock ring |
| `height` | number | 4.4 | tower shaft height (spire adds ≈ 2.2) |

## Animations

`animations: none` — static asset.
