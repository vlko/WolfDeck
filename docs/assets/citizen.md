# citizen

Little standing fox townsperson from the city sheets — fills sidewalks with
life.

## Look

Cream boxy body with straw-gold legs and head; pointy triangular ears,
charcoal dot eyes (auto-blink), blush-pink cheeks, tiny nose; short arms at
the sides. Blob shadow.

## Dimensions & origin

≈ 0.6 wide × 1.5 tall. Origin at ground-center.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | blink timing, jitter |
| `color` | color | `strawGold` | fur color |

## Animations

| name | description |
| ---- | ----------- |
| `idle` | breathing, gentle head tilt, auto-blinks (default) |
| `none` | frozen |
