# hut

Tall conical thatched hut — the golden teepee-like structure from the
reference sheet.

## Look

- Five overlapping faceted straw cone tiers (`strawGold`, darkening upward),
  9 segments each with jittered rims, each tier rotated a seeded amount.
- Three crossed topknot sticks (`deepBrown`) at the peak.
- Dark half-circle doorway at the base.
- Blob shadow.

## Dimensions & origin

≈ 3.4 wide at the base × 4.3 tall (with topknot). Origin at ground-center.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | tier rotation, rim jitter, tint drift |
| `color` | color | `strawGold` | thatch color |
| `height` | number | 3.6 | cone height (topknot adds ~0.7) |

## Animations

`animations: none` — static asset.
