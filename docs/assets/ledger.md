# ledger

Open ledger book lying flat, columns of entries visible.

## Look

- Two `deepBrown` cover halves over a `deepBrown` spine strip, tilted into
  a shallow V (valley at the spine).
- `parchment` page blocks with a `paperWhite` top page on each half.
- Entry columns on each page: a `strawGold` header rule, then five rows of
  `charcoal` description lines (seeded lengths) by the spine and short
  `charcoal` amount figures at the outer edge.
- `dustyRose` bookmark ribbon draped over the front edge.
- Blob shadow.

## Dimensions & origin

≈ 0.62 wide (open) × 0.12 tall × 0.47 deep. Origin at ground-center; the
spine runs along z.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | entry line lengths, facet jitter |
