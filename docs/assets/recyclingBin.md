# recyclingBin

Wheeled kerbside recycling bin — line up a few in different colors for a
sorting station.

## Look

Tapered square bin body with a flat, slightly overhanging lid and a
charcoal stub handle; two charcoal back wheels; small paper-white sorting
label on the front. Body and lid take the stream color. Blob shadow.

## Dimensions & origin

≈ 0.5 wide × 0.7 tall. Origin at ground-center.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | deterministic color pick + jitter |
| `color` | color | seeded from `duckEggBlue`/`sunGold`/`sage`/`dustyRose` | picks the waste stream |
