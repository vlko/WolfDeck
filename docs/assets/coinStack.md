# coinStack

Piles of gold coins — treasury filler that reads as "budget" at a glance.

## Look

2–4 piles (count from the seed) of faceted 7-sided `strawGold` coins with
per-coin offset and rotation jitter, plus a couple of loose coins spilled
flat beside the piles. Blob shadow.

## Dimensions & origin

≈ 1.0 wide × `height` tall (default 0.7); the tallest pile reaches the
full `height`, the rest vary. Origin at ground-center.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | pile count, placement, coin jitter |
| `height` | number | 0.7 | tallest pile height in world units |

## Animations

`animations: none` — static asset.
