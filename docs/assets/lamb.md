# lamb

The sheep's little one — same construction at 58 % scale with baby
proportions.

## Look

Like `sheep`, but:

- proportionally **bigger head** relative to the body,
- a **curly wool cap** — three small faceted spheres on top of the head,
- shorter stubbier legs.

## Dimensions & origin

≈ 1.2 wide × 1.2 tall × 0.9 deep. Origin at ground-center.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | variation (internally offset from sheep so a pair never matches) |
| `color` | color | `paperWhite` | wool color |
| `animation` | string | `idle` | see below |

## Animations

Same set as `sheep`: `idle` (default), `graze`, `blink`, `none`.
