# deskLamp

Small warm table lamp — angled arm, straw-gold cone shade, glowing bulb
with a soft halo. Stack it on a desk or meeting table via prop `y`.

## Dimensions & origin

≈ 0.6 wide × 0.95 tall. Origin at base-center (place at the surface height
of whatever it stands on).

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |
| `shadeColor` | color | `strawGold` |

## Animations

| name | description |
| ---- | ----------- |
| `glow` | halo breathes and flickers like a warm bulb (default) |
| `none` | static |
