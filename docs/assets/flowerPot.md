# flowerPot

Little tan pot with two faceted octahedron blossoms (mauve/white with gold
centers) and leaf cones.

## Dimensions & origin

≈ 0.5 wide × 1.1 tall. Origin at ground-center.

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |
| `colors` | [color] | `[roseMauve, paperWhite]` |

## Animations

| name | description |
| ---- | ----------- |
| `sway` | blossoms nod gently on their stems (default) |
| `none` | still |
