# wasteBasket

Open paper bin with a crumpled paper wad on the floor beside it — someone
missed the shot.

## Dimensions & origin

≈ 0.5 wide × 0.5 tall (plus the wad). Origin at bin ground-center.

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |
| `color` | color | `barkBrown` |

## Animations

`animations: none` — static asset.
