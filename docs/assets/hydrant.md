# hydrant

Tiny muted-red fire hydrant — sidewalk seasoning.

## Look

Faceted cylinder body with a cone dome and two side nubs, in a dusty
brick-rose tone. Blob shadow.

## Dimensions & origin

≈ 0.35 wide × 0.7 tall. Origin at ground-center.

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |
| `color` | color | `#b0777c` |

## Animations

`animations: none` — static asset.
