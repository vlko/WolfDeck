# haystack

Golden straw dome on a little dirt mound, per the reference sheet.

## Look

- Mauve-brown dirt plinth (`roseMauve`), 9-sided.
- Four stacked straw bands (`strawGold`) narrowing upward, each rotated a
  seeded amount so the facets never align, capped with a small cone.
- Jagged straw skirt cone draped over the plinth edge.
- Blob shadow.

## Dimensions & origin

≈ 2.6 wide × 2.6 tall. Origin at ground-center.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | band rotation, tint drift |
| `color` | color | `strawGold` | straw color |

## Animations

`animations: none` — static asset.
