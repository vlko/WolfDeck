# shopBuilding

The corner shop from the reference sheet — cream walls under a big striped
scalloped awning.

## Look

- Cream body with a **duck-egg skirting band** at the base.
- Dark plank door with a knob; four-pane charcoal window in a white frame.
- Signature **awning**: sloped roof of alternating sage/paper-white stripes,
  each stripe ending in a hanging half-disc scallop.

## Dimensions & origin

≈ 5 wide (awning overhang) × 3.6 tall × 2.3 deep at defaults. Origin at
ground-center.

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |
| `width` | number | 4.2 |
| `wallColor` | color | `cream` |
| `awningColor` | color | `sage` |
| `stripeColor` | color | `paperWhite` |
