# bus

The city bus — long duck-egg body with a cream upper and a row of windows.

## Look

Long rounded lozenge (extruded profile with rounded corners): duck-egg
lower body, cream upper band with four dark windows and a door up front,
headlight at the nose, four charcoal wheels with stone-gray hubs. All
detail is mirrored on **both sides**, so the bus reads correctly in either
driving direction. Blob shadow.

## Dimensions & origin

≈ 3.4 long × 1.7 tall. Origin at ground-center; place on a road lane.

## Options

Same as [`car`](car.md): `seed`, `color` (default `duckEggBlue`), `path`,
`speed` (default 2.4), `startAt`, `direction`.

## Animations

| name | description |
| ---- | ----------- |
| `drive` | ping-pongs along `path`, wheels spin, turns at the ends (default) |
| `none` | parked |
