# officePlant

Faceted geometric pot with a fan of pointed agave-like leaves, per the
reference sheet.

## Dimensions & origin

≈ 0.9 wide × 1.5 tall. Origin at ground-center of the pot.

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |
| `color` | color | `pine` (leaves) |
| `potColor` | color | `slateGray` |

## Animations

| name | description |
| ---- | ----------- |
| `leafSway` | the leaf fan turns and rocks very gently (default) |
| `none` | static |
