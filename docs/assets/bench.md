# bench

Park bench for the city sidewalk.

## Look

Three plank-brown seat slats, three backrest slats rising behind, deep-brown
side legs and back posts. Blob shadow.

## Dimensions & origin

≈ 1.7 wide × 1.3 tall × 0.6 deep. Origin at ground-center.

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |
| `color` | color | `plankBrown` |

## Animations

`animations: none` — static asset.
