# paperStack

A small pile of paper sheets, slightly askew. Floor prop or desk prop
(stack via `y`).

## Dimensions & origin

≈ 0.55 × 0.15 × 0.7. Origin at base-center.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | sheet count (3–5), offsets, rotations |

## Animations

`animations: none` — static asset.
