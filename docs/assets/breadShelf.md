# breadShelf

Two-level wooden shelf: upright straw-gold baguettes on top, round tan
loaves below.

## Dimensions & origin

≈ 1.5 wide × 1.7 tall × 0.6 deep. Origin at ground-center.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | baguette lean |

## Animations

`animations: none` — static asset.
