# blackboard

Kid-sized standing chalkboard for the schoolyard — the dark sibling of the
office `whiteboard`.

## Look

`barkBrown` wood frame around a dark slate surface (`charcoal`) with light
chalk scribbles (paper-white strokes on a seeded canvas texture). A-frame
easel: splayed side legs behind the board, rear kickstand leg, chalk tray in
front with a tiny chalk stick. Blob shadow.

| variant | chalk drawing |
| ------- | ------------- |
| `lines` | four wavy rows of chalk "writing" |
| `figures` | sun with rays, little house, stick figure |

## Dimensions & origin

≈ 1.2 wide × 1.4 tall × 0.6 deep (legs). Origin at ground-center.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | scribble shapes/placement, facet jitter |
| `variant` | string | seeded | `lines` \| `figures` |

## Animations

`animations: none` — static asset.
