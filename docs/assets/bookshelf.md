# bookshelf

Parchment case with shelves of colorful seeded book spines — some leaning.

## Look

Back panel + sides + shelf boards in parchment; books in muted palette
colors (dusty rose, sage, duck-egg, straw gold, rose mauve, pine, bark)
with seeded widths/heights, occasional tilted book and gap.

## Dimensions & origin

≈ 1.5 wide × 2.3 tall (3 shelves) × 0.5 deep. Origin at ground-center.

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |
| `color` | color | `parchment` |
| `shelves` | number | 3 |

## Animations

`animations: none` — static asset.
