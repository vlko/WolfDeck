# whiteboard

Cream board on wooden easel legs with seeded pencil scribbles — the meeting
already happened.

## Dimensions & origin

≈ 1.6 wide × 2.3 tall. Origin at ground-center of the legs.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | scribble pattern, leg jitter |

## Animations

`animations: none` — static asset.
