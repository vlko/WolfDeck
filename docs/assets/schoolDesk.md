# schoolDesk

Old-style pupil desk with an attached bench — line up a few facing the
`blackboard`.

## Look

Gently slanted `tanBrown` top with a front panel, `deepBrown` side slab
legs joined to the bench supports by floor runners, attached tan bench
seat behind. A seeded book (dusty-rose / sage / duck-egg) lies askew on the
top. Blob shadow.

## Dimensions & origin

≈ 1.0 wide × 0.65 tall × 0.85 deep (desk front to bench). Origin at
ground-center; the sitter faces +z.

## Options

| option | type | default | notes |
| ------ | ---- | ------- | ----- |
| `seed` | number | 0 | book color/placement, facet jitter |

## Animations

`animations: none` — static asset.
