# marketStall

Brick-based market stall with a counter, side posts and a flat canopy —
dressed with a basket of produce and a little register.

## Dimensions & origin

≈ 3.7 wide × 2.8 tall × 1.7 deep. Origin at ground-center. Counter top ≈
y 1.2 (× scale) if you want to stack more goods on it.

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |

## Animations

`animations: none` — static asset.
