# mug

Steaming coffee mug. Stack it on a `desk` (top ≈ y 1.05 × scale) or
`meetingTable` (top ≈ y 1.07 × scale).

## Look

Parchment cup with a torus handle, dark coffee disc, and three translucent
paper wisps of steam rising and fading on a loop.

## Dimensions & origin

≈ 0.25 wide × 0.2 tall (steam rises to ~0.7). Origin at base-center.

## Options

| option | type | default |
| ------ | ---- | ------- |
| `seed` | number | 0 |
| `color` | color | `parchment` |

## Animations

| name | description |
| ---- | ----------- |
| `steam` | wisps rise, wobble and fade in a loop (default) |
| `none` | cold coffee |
